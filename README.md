## Composition 練習
> 參考課程範例，挑戰 Kata 練習！

> 什麼是 Kata 練習
> 「Kata」是日本字「形」的意思，中文的意思是「套路」。而 Code Kata 就是透過一些小型的程式題目進行重複鍛鍊，就像學功夫套拳。不需要 GUI 或複雜的軟體框架，只需要程式語言本身和一個單元測試函式庫，透過全然遵守 TDD 的規範來練習這些題目，內化及提升程式設計能力。
